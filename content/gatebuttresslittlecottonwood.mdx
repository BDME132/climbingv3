# Gate Buttress Little Cottonwood Climbing Guide

Gate Buttress offers some of the most accessible traditional climbing in Little Cottonwood Canyon, with clean quartzite that's perfect for learning gear placement and multi-pitch skills. The crag sits right off the main road, making it a go-to spot for quick sessions and introducing friends to trad climbing. Note that Little Cottonwood has seasonal closures for bird nesting, so check current access before heading up.
